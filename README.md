# 📊 PriceWatch — مراقب أسعار المنافسين (Demo)

نسخة تجريبية من أداة مراقبة أسعار المنافسين، مبنية بـ Python + Streamlit.

> ⚠️ **هذه نسخة Demo للعرض فقط.** البيانات هنا محاكاة عشوائية لتوضيح شكل النتائج وتجربة الاستخدام. النسخة الكاملة (المدفوعة) تسحب أسعار حقيقية من روابط منتجات فعلية.

---

## 🎯 الفكرة

بدل ما تفحص أسعار منافسيك يدوياً كل يوم، PriceWatch يفحصها لك، يقارنها بالفحص السابق، وينبهك عند أي تغيّر.

## ✨ الميزات (نسخة Demo)

- عرض قائمة منتجات تجريبية مع أسعار محاكاة
- مقارنة تلقائية بين الفحص الحالي والسابق
- تصنيف كل منتج: ارتفع / انخفض / ثابت
- معاينة شكل إشعار التنبيه (بدون إرسال فعلي)
- واجهة عربية كاملة (RTL) بخط Cairo

## 🛠️ التقنيات

- Python 3.10+
- Streamlit
- Pandas

## 🚀 التشغيل محلياً

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ☁️ النشر على Streamlit Cloud

1. ارفع المشروع على GitHub (repo عام أو خاص)
2. ادخل [share.streamlit.io](https://share.streamlit.io)
3. اربط حسابك بـ GitHub واختر الـ repo
4. حدد `app.py` كملف رئيسي
5. اضغط Deploy

## ☁️ النشر على Railway

1. أنشئ مشروع جديد على [railway.app](https://railway.app)
2. اربط الـ GitHub repo
3. أضف Start Command:
   ```
   streamlit run app.py --server.port $PORT --server.address 0.0.0.0
   ```
4. Railway هيكتشف `requirements.txt` وينصّب المكتبات تلقائياً
